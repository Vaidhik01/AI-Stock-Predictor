# 🚀 AI-Powered Multi-Asset Investment Advisor

An intelligent financial platform that helps users make data-driven investment decisions across multiple asset classes including stocks, fixed deposits (FD), commodities, and real estate.

---

## 📌 Problem Statement

Investors often struggle with fragmented financial data and lack a unified system to analyze different investment options. Most tools focus on a single asset class and do not provide predictive insights, risk evaluation, or personalized recommendations.

This project solves that by combining machine learning, sentiment analysis, and financial modeling into one platform.

---

## 💡 Features

### 📈 Stock Analysis
- AI-based stock trend prediction (BUY / SELL / HOLD)
- Technical indicators (MA10, MA50)
- Interactive price charts

### 🧠 Sentiment Analysis
- Analyzes market sentiment to enhance predictions
- Helps understand market psychology

### ⚠️ Risk Analysis
- Volatility-based risk classification (Low / Medium / High)

### 💰 Investment Advisor
- Suggests stocks based on user investment amount
- Supports risk-based portfolio selection

### 📊 SIP Calculator
- Calculates future value of monthly investments
- Visualizes profit vs invested amount

### 🏦 FD Interest Rates
- Displays current FD rates of major banks
- Helps compare low-risk investment options

### ⚖️ FD vs Stocks Comparison
- Compares returns between FD and stock investments
- Visual representation using charts

### 🪙 Gold & Silver Tracker
- Displays commodity prices in ₹ per gram
- Helps diversify investment decisions

### 🏡 Real Estate Analyzer
- Suggests investment areas based on city and budget
- Shows growth potential and risk level

### 📂 Category-Based Filtering
- Filters stocks by sectors (IT, Banking, Energy)

---

## 🛠️ Tech Stack

- Frontend: Streamlit  
- Backend: Python  
- Machine Learning: Scikit-learn (Random Forest)  
- Data Source: yfinance  
- Visualization: Plotly, Streamlit charts
